Документация API Погоды

Этот API предоставляет доступ к данным о погоде. Здесь вы найдете документацию по методам, параметрам запросов и форматам ответов.

Авторизация

Для доступа к API необходимо получить токен авторизации. Используйте приведенные ниже шаги для получения токена и его использования в последующих запросах.

  • Тестовый пользователь: { "email": "test@example.com", "password": "testtest" }

Получение токена

Для получения токена отправьте запрос на конечную точку /api/token с параметрами email и password в теле запроса:

URL: /api/token

Метод: POST

Описание: Возвращает токен авторизации.

Тело запроса:

{
    "email": "test@example.com",
    "password": "testtest"
}

Пример ответа:

{
    "success": true,
    "token": "test_3d156eb7e37312d4196650dedb387ff4e8cf97dc78636cc02ed68a716a266a30"
}

Использование токена

После получения токена добавьте его в запросы к API в качестве параметра token в строке запроса.

Пример запроса с токеном

GET http://example.com/api/weather/current?token=test_3d156eb7e37312d4196650dedb387ff4e8cf97dc78636cc02ed68a716a266a30

Описание параметров

  • token (обязательный) — токен доступа, полученный из ответа на запрос /api/token.

Методы

1. Получить все данные о погоде

URL: /api/weather

Метод: GET

Описание: Возвращает все доступные данные о погоде.

Пример запроса:

curl -X GET "http://example.com/api/weather?token={token}"

Пример ответа:

[
  {
    "id": 1,
    "time": "2024-06-01 08:00:00",
    "air_temperature_noaa": 25.3,
    "air_temperature_sg": 26.1,
    "humidity_noaa": 70,
    "humidity_sg": 65,
    "pressure_noaa": 1015,
    "pressure_sg": 1016,
    "wind_direction_noaa": 180,
    "wind_direction_sg": 185,
    "wind_speed_noaa": 10,
    "wind_speed_sg": 11
  }
  ...
]

Описание параметров ответа:

Параметр Тип Описание
id Integer Уникальный идентификатор записи
time String Время записи данных (формат YYYY-MM-DD HH:MM:SS)
air_temperature_noaa Float Температура воздуха по данным NOAA
air_temperature_sg Float Температура воздуха по данным SG
humidity_noaa Integer Влажность воздуха по данным NOAA (%)
humidity_sg Integer Влажность воздуха по данным SG (%)
pressure_noaa Integer Атмосферное давление по данным NOAA (hPa)
pressure_sg Integer Атмосферное давление по данным SG (hPa)
wind_direction_noaa Integer Направление ветра по данным NOAA (градусы)
wind_direction_sg Integer Направление ветра по данным SG (градусы)
wind_speed_noaa Float Скорость ветра по данным NOAA (м/с)
wind_speed_sg Float Скорость ветра по данным SG (м/с)

Структура данных

Основные поля в ответах:

Поле Тип Описание
id Integer Уникальный идентификатор записи
time String Время записи данных (формат YYYY-MM-DD HH:MM:SS)
air_temperature_noaa Float Температура воздуха по данным NOAA
air_temperature_sg Float Температура воздуха по данным SG
humidity_noaa Integer Влажность воздуха по данным NOAA (%)
humidity_sg Integer Влажность воздуха по данным SG (%)
pressure_noaa Integer Атмосферное давление по данным NOAA (hPa)
pressure_sg Integer Атмосферное давление по данным SG (hPa)
wind_direction_noaa Integer Направление ветра по данным NOAA (градусы)
wind_direction_sg Integer Направление ветра по данным SG (градусы)
wind_speed_noaa Float Скорость ветра по данным NOAA (м/с)
wind_speed_sg Float Скорость ветра по данным SG (м/с)

Примечания

  • Формат времени в запросах и ответах должен быть в соответствии с UTC.
  • Параметры температур и скоростей ветра указаны в метрической системе (градусы Цельсия и метры в секунду).
  • Для всех запросов к API необходимо предоставить корректный авторизационный токен. Без токена доступ к API будет ограничен.
